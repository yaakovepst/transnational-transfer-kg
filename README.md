# Transnational Transfer LPG

An article-to-LPG and wiki prototype based on one primary source document.

## Live demo

https://yaakovepst.github.io/transnational-transfer-kg/

## What this is

This repository contains a static labeled property graph (LPG) and linked wiki demo. It models people, organizations, funders, publications, events, ideas, places, and transnational transfer relationships from the source document.

The source scholarship belongs to Segal and Greenspan. This repository demonstrates Jacob Epstein’s contribution: article-to-LPG modeling, interface design, wiki-page generation from graph relationships, internal wiki navigation, and iterative editorial review.

The project was developed through an AI-with-human-in-the-loop workflow (Mosqueira-Rey et al., 2023), with AI-assisted extraction, modeling, and wiki generation reviewed and revised through human editorial judgment.

Open `docs/index.html` locally, or use the GitHub Pages live demo link above.

The demo uses Cytoscape.js and fCoSE through CDN links, so internet access is required.
