# \# Transnational Transfer LPG

# 

# A static article-to-LPG and wiki prototype based on one primary source document:

# 

# Segal, A. A., \& Greenspan, I. (2024). The Americanization of Israeli Conservative Civil Society: A critical community and transnational transference perspective. \*Israel Studies, 29\*(2), 144–169.

# 

# This project transforms the article into an interactive labeled property graph (LPG) and linked wiki demo. It models people, organizations, funders, publications, events, ideas, places, and transnational transfer relationships.

# 

# The scholarly source, argument, and empirical discussion belong to Segal and Greenspan. This repository is a derivative portfolio project that demonstrates Jacob Epstein’s contribution: article-to-LPG modeling, interface design, wiki-page generation from graph relationships, internal wiki navigation, and iterative editorial review.

# 

# The project was developed through an AI-with-human-in-the-loop workflow (Mosqueira-Rey et al., 2023). AI-assisted extraction, modeling, and wiki generation were reviewed and revised through human editorial judgment.
