# Limitations

This project is a prototype based on one primary source document:

Segal, A. A., & Greenspan, I. (2024). The Americanization of Israeli Conservative Civil Society: A critical community and transnational transference perspective. *Israel Studies, 29*(2), 144–169.

It should not be read as a complete map of Israeli conservatism, American conservative influence, Israeli civil society, or transnational political transfer. It is an LPG and wiki representation of one article’s argument and source base.

The graph is interpretive. Transforming prose into nodes, edges, and metadata requires modeling choices. Some choices are straightforward, such as representing a named organization as an organization node. Other choices are interpretive, such as deciding whether a broad descriptor should be modeled as a node, metadata tag, or excluded from the graph.

The project was developed through an AI-with-human-in-the-loop workflow (Mosqueira-Rey et al., 2023), but human review does not make the graph error-free. The workflow was designed to reduce mechanical extraction errors, improve prose, and avoid unsupported overclaiming. Remaining errors or omissions are possible.

The demo is not an official dataset from Segal and Greenspan, *Israel Studies*, or any organization represented in the graph. It is a portfolio project showing how a natural-language source document can be transformed into a static, navigable LPG and wiki.

This version does not include private development files, flagged review artifacts, or internal drafting notes. Reader-facing pages are meant to show the final artifact, not the full development history.
